## Vim

Vim是一种模态编辑器，有几种模式
  - 插入模式，您在编辑器中输入文本并将其提交给文档；
  - 正常模式，您通过键盘输入参数以执行各种功能，包括：在文档周围移动光标、搜索和操作文档中的文本（例如剪切和粘贴）
  - 命令模式：在 shell 中输入命令的命令模式
  - 可视模式：选择要操作的文本。
