## http 

### http 状态码

- 1xx: 表示目前是协议处理的中间状态，还需要后续操作。
- 2xx: 表示成功状态。
- 3xx: 重定向状态，资源位置发生变动，需要重新请求。
- 4xx: 请求报文有误。
- 5xx: 服务器端发生错误。


## 3次握手、4次挥手

握手目的，确认双方的发送能力、接收能力是否正常。

每个阶段各确认了什么：

第一次握手：客 => 服，客户端发送能力正常

第二次握手：服 => 客，+ 服务端接收能力、服务端发送能力，客户端接收能力（但是服务端不知道）

第三次握手：客 => 服，+ 客户端接收能力（服务端知道）
