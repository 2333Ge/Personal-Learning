# 快捷键

快捷键[官网链接][official]

|    快捷键    |                    描述                    | 备注  |
| :----------: | :----------------------------------------: | :---: |
| option+cmd+L |                 格式化代碼                 |       |
|   shift+`    |            关闭最近打开的操作栏            |       |
|   shift+f6   |                 变量重命名                 |       |
|  ctrl+alt+h  |               查看方法调用栈               |       |
|    cmd+y     | 查看源码时不跳转显示参数或某个子函数的源码 | ![a1] |
