左右切换窗口

$ ctrl + 左/右

最近两个应用切换

短按

$  command + tab

应用窗口切换

长按，再按 tab 切换 // 下一个窗口

$  command + tab

// 上一个窗口

// 如浏览器，vscode

$ command + shift + [或]
